# Corolla Maintenance Hub

Complete maintenance guide for Toyota Corolla 2011 (1.8L 2ZR-FE engine)

## Features
- 📊 Mileage tracker with localStorage
- 📋 Interactive maintenance checklist
- 📺 YouTube tutorial links
- 🎨 Netflix-style dark theme
- 📱 Responsive design

## Vehicle
- Model: Toyota Corolla 2011
- Engine: 1.8L 2ZR-FE
- Origin: Made in Japan

## How to Use
1. Open index.html in browser
2. Log your current mileage
3. Check off completed maintenance tasks
4. Follow YouTube tutorials for each system
